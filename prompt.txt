Nome: Coiled Spring Strategy App
Tipo: SaaS web app per trader di opzioni
Obiettivo: tool di supporto decisionale per selezione, 
salvataggio, monitoraggio e gestione di opzioni LEAPS 
su mercato USA secondo una logica antifragile, long Vega, 
risk-defined, orientata alla compressione della volatilità.

Il prodotto NON è uno scanner generico.
È una macchina di optionality disciplinata che aiuta 
l'utente a scartare ciò che non merita capitale.

Stack target deciso:
- Backend: Python + FastAPI
- Frontend: React + TypeScript / Next.js
- Database: PostgreSQL
- Broker abstraction layer (primo broker: IBKR via ib_insync)
- Auth: JWT
