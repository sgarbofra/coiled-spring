Nel file backend-python/app/routers/ crea un nuovo file market.py

Aggiungi un endpoint GET /api/market/quote?tickers=AAPL,SPY,NVDA

L'endpoint deve:
1. Ricevere una lista di ticker separati da virgola
2. Per ogni ticker chiamare yfinance: yf.Ticker(ticker).fast_info
3. Restituire JSON con: ticker, currentPrice, previousClose, 
   marketCap, currency
4. Gestire errori per ticker non trovati
5. Richiedere autenticazione con get_current_user

Registra il router in main.py con prefix /api/market
